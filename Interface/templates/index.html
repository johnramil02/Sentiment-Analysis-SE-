About Project SAWE

Project Sentiment Analysis with Emoji

Project SAWE is a web-based sentiment analysis application that uses Convolution Neural Network to classify sentiments.

1

Click "Analyze Sentiments"

This button will redirect you to the page where this application will analyze your desired entered or uploaded sentiments.

2

Options

After redirected, you will choose in the "Analyze Your Sentiments" web page to choose between a single or multiple sentiments

3

Single Sentiment

This is where you will enter your sentiment and use the analyze button to see the result of your classified sentiment

4

Multiple Sentiments

This is where you will upload your sentiment csv file and will result a summary of your uploaded file and a csv file with classified sentiment

Note:

Features

Project SAWE can only accomodate Tagalog, English or Taglish languages only and different emojis that expresses emotions
